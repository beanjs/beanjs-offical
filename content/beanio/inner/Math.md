### Math.wrap(x,max)

::i-chinese{sha="6d9ccd9670e9ebfbbe10223c1858b85b0c0bb0325d45693632ed4cc1d13545a8"}
::
DEPRECATED - This is not part of standard JavaScript libraries

instance you might do: 
```javascript

```
